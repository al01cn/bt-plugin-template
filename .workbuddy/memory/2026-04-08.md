# 2026-04-08

## pack.py 重构：去掉 .tpl 模板机制
- **src/ 全部是源码**，开发者直接编辑，不生成中间文件
- 打包时：读取 src/ 源码 → 替换 `{{#key#}}` 占位符 → 写入 zip，**不修改 src/ 任何文件**
- 只有实际包含占位符的文件才做替换（`has_placeholder()` 检测），其他文件原样 `zf.write`
- `info.json` 直接从 `package.json` 构建写入 zip，不依赖源码文件
- 删除了所有 `.tpl` 文件，gitignore 里去掉了生成文件规则

## 占位符使用约定
- 代码文件（JS/Python）：写 `"{{#key#}}"` 字符串形式，避免代码审查报错
- HTML/文本文件：直接写 `{{#key#}}`
- 替换正则：`\{\{#(\w+)#\}\}`

## 其他改动
- `package.js`：`var __BT_PLUGIN__` + `Object.freeze()` 冻结，防篡改（不再用 `window.__BT_PLUGIN__`）
- `main.js`：精简为 request 封装 + hello demo + init
- `index.html`：极简 demo 页面（logo + 标题 + Hello World 按钮）
